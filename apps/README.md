# Apps

SDKWork AIoT client application roots.

| App root | Architecture | Purpose |
| --- | --- | --- |
| `apps/sdkwork-aiot-pc/` | PC React | Browser/desktop operator console |
| `apps/sdkwork-aiot-h5/` | H5 mobile React | Phone-first mobile console |
| `apps/sdkwork-aiot-mini-program/` | WeChat mini program | Mini program device/IoT surfaces |
| `apps/sdkwork-aiot-shared/` | Shared packages | Cross-client app-core services and SDK wiring |

Server-side Rust services remain under `services/` and `crates/` at the repository root.
